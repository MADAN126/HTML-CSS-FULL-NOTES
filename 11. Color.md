# CSS Colors

CSS provides multiple ways to specify colors.

---

## Named Colors

Use predefined color names.

```css
color: black;
color: red;
color: blue;
color: green;
color: white;
```

Example:

```css
h1{
    color: red;
}
```

---

## Hexadecimal Colors

Format:

```css
#RRGGBB
```

- RR → Red
- GG → Green
- BB → Blue

Range:

```text
00 to FF
```

Examples:

```css
color: #ff0000;   /* Red */
color: #00ff00;   /* Green */
color: #0000ff;   /* Blue */
color: #ffffff;   /* White */
color: #000000;   /* Black */
```

Example:

```css
h1{
    color: #00ff00;
}
```

---

## RGB Colors

Format:

```css
rgb(red, green, blue)
```

Range:

```text
0 to 255
```

Examples:

```css
color: rgb(255, 0, 0);     /* Red */
color: rgb(0, 255, 0);     /* Green */
color: rgb(0, 0, 255);     /* Blue */
color: rgb(0, 0, 0);       /* Black */
color: rgb(255, 255, 255); /* White */
```

Example:

```css
h1{
    color: rgb(255, 0, 0);
}
```

---

## RGBA Colors

RGBA adds transparency.

Format:

```css
rgba(red, green, blue, alpha)
```

Alpha value:

```text
0   → Fully transparent
0.5 → 50% transparent
1   → Fully opaque
```

Examples:

```css
color: rgba(255, 0, 0, 0.5);
background-color: rgba(0, 0, 255, 0.3);
```

Example:

```css
h1{
    background-color: rgba(255, 0, 0, 0.5);
}
```

---

## HSL Colors

Format:

```css
hsl(hue, saturation, lightness)
```

### Hue

```text
0°   → Red
120° → Green
240° → Blue
```

Range:

```text
0–360°
```

### Saturation

Controls intensity.

```text
0%   → Gray
100% → Full color
```

### Lightness

Controls brightness.

```text
0%   → Black
50%  → Normal color
100% → White
```

Examples:

```css
color: hsl(0, 100%, 50%);
color: hsl(120, 100%, 50%);
color: hsl(240, 100%, 50%);
```

---

## HSLA Colors

HSLA is HSL with transparency.

Format:

```css
hsla(hue, saturation, lightness, alpha)
```

Examples:

```css
color: hsla(0, 100%, 50%, 0.5);
color: hsla(120, 100%, 50%, 0.7);
```

Example:

```css
h1{
    color: hsla(240, 100%, 50%, 0.8);
}
```

---

## Important Rule

If the same property is declared multiple times:

```css
h1{
    color: red;
    color: blue;
    color: black;
}
```

The browser uses the **last valid declaration**.

Result:

```text
black
```
