# CSS Borders

---

## What is a Border?

A border is the visible line drawn around an HTML element.

Every HTML element is treated as a rectangular box by the browser, and a border defines the edge of that box.

```text
+---------------------------+
|         Margin            |
|  +---------------------+  |
|  |       Border        |  |
|  |  +---------------+  |  |
|  |  |    Padding    |  |  |
|  |  |  +---------+  |  |  |
|  |  |  | Content |  |  |  |
|  |  |  +---------+  |  |  |
|  |  +---------------+  |  |
|  +---------------------+  |
+---------------------------+
```

Border sits:

```text
Outside → Padding
Inside  → Margin
```

---

## Why Do Borders Exist?

Borders solve several problems:

### Visual Separation

Without borders:

```text
Everything blends together.
```

With borders:

```text
Sections become distinguishable.
```

Example:

```css
.card{
    border: 1px solid gray;
}
```

---

### Highlighting Elements

Used for:

- Buttons
- Input fields
- Cards
- Tables
- Alerts
- Images

Example:

```css
button{
    border: 2px solid blue;
}
```

---

### Debugging Layouts

Developers often use borders temporarily:

```css
*{
    border: 1px solid red;
}
```

This reveals element boundaries.

---

# Border Syntax

Basic syntax:

```css
border: width style color;
```

Example:

```css
border: 2px solid black;
```

Equivalent to:

```css
border-width: 2px;
border-style: solid;
border-color: black;
```

---

# Border Properties

## 1. border-width

Controls thickness.

Syntax:

```css
border-width: value;
```

Example:

```css
border-width: 5px;
```

Output:

```text
5-pixel thick border.
```

---

### Different Widths

```css
border-width: 1px;
border-width: 5px;
border-width: thick;
border-width: medium;
border-width: thin;
```

---

## Multiple Values

### Four Values

```css
border-width: 1px 2px 3px 4px;
```

Meaning:

```text
Top    → 1px
Right  → 2px
Bottom → 3px
Left   → 4px
```

---

### Three Values

```css
border-width: 1px 2px 3px;
```

```text
Top    → 1px
Right  → 2px
Left   → 2px
Bottom → 3px
```

---

### Two Values

```css
border-width: 1px 2px;
```

```text
Top    → 1px
Bottom → 1px
Right  → 2px
Left   → 2px
```

---

# 2. border-style

Defines how the border looks.

Syntax:

```css
border-style: value;
```

---

## solid

```css
border-style: solid;
```

```text
────────
```

Continuous line.

Most common.

---

## dotted

```css
border-style: dotted;
```

```text
.........
```

Dots.

---

## dashed

```css
border-style: dashed;
```

```text
- - - - -
```

Broken line.

---

## double

```css
border-style: double;
```

```text
========
```

Two lines.

---

## groove

```css
border-style: groove;
```

3D carved effect.

---

## ridge

```css
border-style: ridge;
```

Raised 3D effect.

---

## inset

```css
border-style: inset;
```

Pressed inward appearance.

---

## outset

```css
border-style: outset;
```

Raised outward appearance.

---

## none

```css
border-style: none;
```

No border.

---

## hidden

```css
border-style: hidden;
```

Used mainly in tables.

Acts similarly to none.

---

# Important Rule

Without border-style:

```css
border-width: 5px;
border-color: red;
```

Nothing appears.

Because:

```text
Style is mandatory.
```

Example:

❌

```css
border: 5px red;
```

✅

```css
border: 5px solid red;
```

---

# 3. border-color

Sets border color.

Syntax:

```css
border-color: color;
```

Example:

```css
border-color: red;
```

---

## Supported Color Formats

Named colors:

```css
border-color: blue;
```

Hex:

```css
border-color: #00ff00;
```

RGB:

```css
border-color: rgb(255,0,0);
```

RGBA:

```css
border-color: rgba(255,0,0,0.5);
```

HSL:

```css
border-color: hsl(240,100%,50%);
```

HSLA:

```css
border-color: hsla(240,100%,50%,0.7);
```

---

# Border Shorthand

Instead of:

```css
border-width: 2px;
border-style: solid;
border-color: black;
```

Use:

```css
border: 2px solid black;
```

Most common approach.

## Right Border

```css

```

---

## Bottom Border

```css

    border-top: 4px solid red;
```

# Border Radius

## What is it?

Rounds corners.

Without radius:

```text
+---------+
|         |
+---------+
```

With radius:

```text
╭─────────╮
│         │
╰─────────╯
```

---

## Syntax

```css
border-radius: value;
```

Example:

```css
border-radius: 10px;
```

---

## Circle

```css
border-radius: 50%;
```

Example:

```css
img{
    border-radius: 50%;
}
```

Creates circular profile pictures.

---

# Different Corner Radius

Four values:

```css
border-radius: 10px 20px 30px 40px;
```

Meaning:

```text
Top Left     → 10px
Top Right    → 20px
Bottom Right → 30px
Bottom Left  → 40px
```

---

# Border and Box Model

Remember:

```text
Total Width =
Content
+ Left Padding
+ Right Padding
+ Left Border
+ Right Border
```

Example:

```css
width: 200px;
padding: 20px;
border: 5px solid black;
```

Actual width:

```text
200 + 20 + 20 + 5 + 5
= 250px
```

Unless:

```css
box-sizing: border-box;
```

is used.

# Common Mistakes

## Forgetting border-style

Wrong:

```css
border: 2px red;
```

Correct:

```css
border: 2px solid red;
```

---

## Excessively Thick Borders

Bad:

```css
border: 20px solid black;
```

Makes UI heavy.

Use subtle borders.

---

## Using Multiple Borders Unnecessarily

Avoid:

```css
border-top: 1px solid;
border-right: 1px solid;
border-bottom: 1px solid;
border-left: 1px solid;
```

Use:

```css
border: 1px solid;
```

unless sides differ.

# Interview Questions

## What is the shorthand syntax for border?

```css
border: width style color;
```

Example:

```css
border: 2px solid black;
```

---

## Which border property is mandatory?

```text
border-style
```

Without it, no border is displayed.

---

## Difference Between Border and Outline?

Border:

```text
Part of box model.
Affects element size.
```

Outline:

```text
Outside the border.
Does not affect size.
```

---

## How do you make a circle using CSS?

```css
border-radius: 50%;
```

Applied to a square element.

# Mental Model

```text
Content
↓
Padding
↓
Border
↓
Margin
```

Border is the visible boundary of an element's box.

It can define structure, improve usability, aid debugging, and enhance visual design.

---

# Summary

| Property | Purpose |
|----------|-----------|
| border | Shorthand for all border settings |
| border-width | Controls thickness |
| border-style | Controls appearance |
| border-color | Controls color |
| border-top/right/bottom/left | Styles individual sides |
| border-radius | Rounds corners |

> Borders define the visible edges of an element, making interfaces structured, understandable, and visually appealing.
